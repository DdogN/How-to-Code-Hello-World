html

What the html tag does is basically tells you that you are starting the code. it is also telling you that you are coding in html and NOT in something like javascript or python.

body

A body tag is a tag that tells you where most of your code starts and ends. So you you put one at the top of the page that tells you that you are starting the code which is what this section is about. Inside of the body tag is where you put your code, there are many different tags that you can also put inside of a body tag that tell you where writing, images and code will be. The difference between a body tag and a head tag is that the head tag isn't actually on the website but is a description of the website that will be shown in your browser.

h1

The h1 tag is something that goes inside of the body tag. It is a Heading size 1 tag, therefor the name h1. There is also different types of heading tags like h2 h3 h4 and h5. What the different numbers after them mean is the different sizes of font. But if you are a beginning coder you should just stick to h1 because it is the biggest and easiest to code with

/h1

You may be wondering why there is another h1 but with a slash before it.and we will start with why there is a slash before it, the slash means that you are ending the function (telling the computer to stop.) you MUST have a end tag after every start tag otherwise the computer won't know to stop there for your code wont show up and the rest of your code after the unended function will not work.

/body

A slash body tag is saying that you are done coding the body. It usually is one of the last things on the page being that it ends almost all of your code.

/html

A /html tag is a simple thing that just ends your code. There is really nothing else to it other than that
